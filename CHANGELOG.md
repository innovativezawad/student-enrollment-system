# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-01-21

### Added
- Initial project setup with React + TypeScript + Vite
- Supabase integration for authentication and database
- Dashboard with statistics (total students, revenue, due amounts)
- Student management (CRUD operations)
- Student list with search and filter functionality
- Payment tracking (full payment and installment support)
- Authentication with protected routes
- Toast notifications for user feedback
- Modal forms for add/edit operations
- Responsive UI design with Tailwind CSS
- GitHub Pages deployment configuration
- Documentation (README, SETUP, DEPLOYMENT, CONTRIBUTING)

### Features
- ✅ Admin login/logout
- ✅ Dashboard statistics
- ✅ Add/edit/delete students
- ✅ Search students by name/phone
- ✅ Filter students by course
- ✅ Track full payments
- ✅ Track installment payments
- ✅ Payment history
- ✅ Responsive design
- ✅ Toast notifications

### Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router
- Lucide React
